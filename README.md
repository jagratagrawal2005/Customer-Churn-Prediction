# AI-Powered Customer Churn Prediction

## Overview

This project predicts whether a telecom customer is likely to churn using Machine Learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and business insights to help improve customer retention.

---

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Customer Retention Analysis
* Multiple Machine Learning Models

  * Logistic Regression
  * Random Forest
* Hyperparameter Tuning
* Cross Validation
* Confusion Matrix
* Classification Report
* Feature Importance Analysis
* Business Summary & Insights

---

## Dataset

**IBM Telco Customer Churn Dataset**

The dataset contains customer demographic information, subscribed services, account details, and churn status.
